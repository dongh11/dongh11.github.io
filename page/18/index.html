Data Types in R

Data types include

  • logical
  • numeric
  • integer
  • complex
  • character
  • raw

The simplest of these objects is the vector object and there are six data types of these atomic vectors, also termed as six classes of vectors. The other R-Objects are built upon the atomic vectors.

  • vectors
  • lists
  • matrices
  • arrays
  • factors
  • data frames
